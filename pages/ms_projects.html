  • Understanding Synthetic Politics on Reddit

    View on Github

    This project investigates how AI-generated political content (deepfakes) is shared, discussed, and moderated on Reddit. Using a mixed-method approach combining large-scale automated analysis with qualitative coding, we analyze posts from two subreddits — r/politics and r/futurology — spanning December 2022 through February 2026. The study covers the period immediately following the public release of ChatGPT (November 2022), capturing the full arc of generative AI's integration into political discourse online.

  • LLMComp: Prompt Compression for Robust Defense Against LLM Jailbreaks

    View on Github

    We present LLMComp, a prompt-compression–based defense mechanism designed to improve robustness against Large Language Model (LLM) jailbreak attacks in both monolingual and multilingual settings. This work is built upon and extends SecurityLingua (Li et al.), which is adopted as a baseline defense mechanism and further evaluated in multilingual jailbreak scenarios.

  • Deep Learning Network Steganography

    View on Github

    We implementated Deep Learning Network Steganography, which investigates how a complete neural network can be concealed within another neural network while preserving the functionality of both models. Inspired by the concept of digital steganography, our approach embeds a secret model inside a stego model such that the stego model continues to perform its original (cover) task, while the hidden model can be accurately recovered using a shared secret key. We explore both intra-task and inter-task settings, propose several filter insertion strategies, and evaluate the effectiveness of our framework on multiple benchmark datasets. Our results demonstrate that deep neural networks can serve as effective carriers for hidden models with minimal impact on cover-task performance.

  • Replication & Extension of IAC-eval benchmark

    View on Github

    I replicated and extended the IaC-Eval benchmark introduced by Kon et al. (NeurIPS 2024), which evaluates the capability of Large Language Models (LLMs) to generate correct, secure, and compliant Infrastructure-as-Code (IaC) configurations from natural language cloud specifications. Building upon the original benchmark, this project investigates the impact of task complexity and advanced prompting strategies on IaC generation quality while expanding the evaluation to 458 AWS Terraform scenarios spanning six levels of difficulty. The study systematically assesses the functional correctness, security, and compliance of LLM-generated Terraform code, providing a comprehensive empirical analysis of current LLM capabilities for cloud infrastructure automation.

  • Regular Expression Game

    View on Github

    We developed an interactive educational web application to help students learn and practice regular expressions (regex) through a gamified learning experience. The platform includes pre- and post-assessments, a multi-level password game, anonymous score collection, and instructor resources for classroom integration. The game was designed for use in COMPSCI 250 at UMass Amherst to support hands-on learning and evaluate student progress.
